# GZB クラス別 ランダム対戦表

バドミントンダブルス用に、A / B / C / D の各グループごとにランダムな対戦表を作成するHTMLアプリです。

## 主な仕様

- 4グループ対応（A / B / C / D）
- 各グループ最大20名
- 1〜3名、7名登録時はエラー表示
- 4〜6名、8〜12名でランダム対戦表を作成
- 各グループは最大2コート運用
- グループごとに1〜3コートで対戦表を作成
- 余りの人数は各試合で「休憩」として表示
- 同じペアや似た対戦が連続しにくいよう、偏りを抑えて作成
- デフォルト試合数は9試合
- 上部ボタンからA・B・C・Dを1枚にまとめたセット画像（PNG）を共有シートへ出力可能
- 各グループごとに対戦表画像（PNG）を個別出力可能
- 名前入力中に即時自動保存。入力・変更・画面離脱時にも保存します。
- 名簿・設定・前回作成した対戦表はブラウザ内に保存

## GitHub Pages 公開手順

1. GitHubで新規リポジトリを作成します。
2. このZIPを展開し、上記ファイル一式をリポジトリ直下へアップロードします。
3. GitHubの `Settings` → `Pages` を開きます。
4. `Build and deployment` の `Source` を `Deploy from a branch` にします。
5. `Branch` を `main`、フォルダを `/root` にして保存します。
6. 数十秒後に表示される GitHub Pages のURLをSafariで開きます。
7. iPhone Safariの共有ボタンから「ホーム画面に追加」を選びます。

## 保存仕様

- 名簿は入力中に即時自動保存されます。名簿、試合数設定、前回作成した対戦表はブラウザの `localStorage` に保存されます。
- 保存データは「同じ端末」「同じブラウザ」「同じURL」で保持されます。
- URL変更や端末変更に備えたい場合は、名簿バックアップを利用してください。

## 対戦表画像について

- 各グループの結果カードにある「画像出力」ボタンから、PNG画像を保存できます。
- 画像は3試合を横一列で配置し、その下に続きの試合を並べるレイアウトです。

## 出力ボタンについて

- `結果のテキスト出力`：作成済み対戦表をテキストとしてコピーします。
- `結果の画像出力`：A・B・C・Dを1枚にまとめたセット画像を共有シートへ渡します。共有非対応環境では保存用画像を表示します。


## 対戦表作成可能人数

- 4名：休憩なし、1コート
- 5名：休憩1名、1コート
- 6名：休憩2名、1コート
- 7名：エラー
- 8名：休憩なし、2コート
- 9名：休憩1名、2コート
- 10名：休憩2名、2コート
- 11名：休憩3名、2コート
- 12名：休憩4名、2コート

- iPhone Safari など Web Share API 対応環境では、画像出力時に共有シートを直接開きます。


## V2 変更仕様

- グループ表記を `A / B / C / D` に変更しました。
- 旧データは `A → A`、`B1 → B`、`B2 → C`、`C → D` の順で移行します。
- 各グループの最大登録人数を20名に変更しました。
- 各グループごとに使用コート数を1〜3コートで指定できます。
- 指定コート数に応じて、各コート名を個別設定できます。
- 指定コート数に対して必要人数が不足する場合はエラー表示します。
  - 1コート：最低4名
  - 2コート：最低8名
  - 3コート：最低12名
- 7名は従来どおりエラー扱いです。
- 組合せロジックは、同じペアだけでなく、同じ相手ペア・同じ相手選手との重複をより強く避ける方向へ調整しています。

- 画像出力は、9試合・1コートの標準レイアウト時に、おおむね 3000×2000px 前後へ収まるよう調整しました。
- 画像内のコート名・ペア名フォントを大きめにし、視認性を改善しました。

- 使用コート数は1〜3のプルダウン選択に変更しました。
- 画像出力時、ペア名カードがCourt枠内に収まるよう縦位置を再調整しました。

- 使用コート数は 1コート / 2コート / 3コート のプルダウン選択です。
- 画像出力は 3000x2000 の上限固定をやめ、見やすさ優先の大きめ出力に変更しました。
- 試合カードの外枠は濃いグリーンの太枠に変更しました。

- 画像出力の試合カード外枠を、より濃いグリーンの太線に調整しました。
- 画像出力のフォントは、中国語簡体字も表示しやすいゴシック系フォントスタックへ変更しました。
- コート名・ペア名はさらに大きくし、白フチと影で視認性を高めています。
