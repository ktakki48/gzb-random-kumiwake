GZB ランダム組み分け

A / B1 / B2 / C の各グループから、最大6名を目安に均等な組をランダム作成します。
名簿はこの端末のブラウザ内に保存されます。

使い方名簿を編集 → 保存 → 「全グループ作成」。4〜6名は1組、8〜12名はできるだけ均等な2組に分けます。
1〜3名と7名は人数エラーになります。

iPhoneでアプリのように使う場合
  • GitHub Pagesで公開したURLをiPhoneのSafariで開きます。
  • 共有ボタン → 「ホーム画面に追加」を選ぶと、アプリ風に起動できます。
  • 名簿保存はlocalStorageを使用します。同じ端末・同じブラウザ・同じURLで保持されます。
  • 機種変更やURL変更に備える場合は「名簿バックアップ」でバックアップしてください。
